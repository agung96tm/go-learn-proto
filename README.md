Go Learn Proto
============================================
Example interaction between Grpc (Server) and Http (Client).

** WORK IN PROGRESS **

### How to run
#### First time?
```shell
make dependencies
make proto_generate
```

#### Run Applications
```shell
make runapi
make runrpc
```
* Access Api: [http://localhost:8000](http://localhost:8000)
* Access Grpc: [grpc://localhost:5000](grpc://localhost:5000)
